PLC Pneumatic Gripper Device

Description

This device operates with a PLC and integrates distribution, transport, and carry functions. The device uses a pneumatic gripper for carrying, ensuring high efficiency and control in automation systems.

Technical Parameters

  • Base Plate: Aluminum Profile
  • Structure: Aluminum Profile
  • Function Indicator: Tower Type (Red, Green, Yellow)
  • PLC: Relay Type
  • PLC Input: 16 Nos
  • PLC Output: 12 Nos
  • Carry Function: Using Gripper method
  • Pneumatic Cylinder: 03 Nos
  • Bore Size: 10mm
  • Stroke Length: 75mm
  • Proximity Sensor: 01 set
  • Pneumatic Gripper: 01 Nos, Long stroke type parallel gripper
  • Solenoid Valve: 4 Nos
  • Cabinet Control Circuit Power Supply: DC24V
  • Temperature: -10°C ~ 40°C
  • Humidity: ≤90%
  • Max Air Pressure: 0.7Mpa
  • Machine Capacity: ≤0.5KVA
  • Input Power: Single-phase, three-wire AC 220V±10% 50Hz/60Hz

Accessories

  • Experimental Manual: 1 copy
  • Connecting Cord: 1 unit
  • Air Compressor: 1 unit
